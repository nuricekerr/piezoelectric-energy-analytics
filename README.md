# 🔬 Piezoelectric Energy Harvesting Data Analytics & ML Prediction

This project focuses on analyzing simulation and experimental data from smart materials (PVDF/PZT) used in energy harvesting and vibration damping. It utilizes Python for engineering data analytics and Machine Learning to predict voltage output based on vibration frequency and acceleration.

---

## 📂 Project Structure

* **`src/`**: Core Python source codes (`data_analysis.py`).
* **`piezo_analysis_chart.png`**: Generated data visualization chart showing the relationship between vibration inputs and voltage outputs.

---

## 🛠️ Technologies & Libraries Used

* **Language:** Python 3.x
* **Data Libraries:** Pandas, NumPy
* **Visualization:** Matplotlib, Seaborn
* **Machine Learning:** Scikit-Learn (Linear Regression, Train-Test Split)
